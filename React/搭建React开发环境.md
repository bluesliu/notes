# 搭建 React 开发环境

## 安装命令行工具

全局安装`create-react-app`命令行工具，这个工具可以快速创建`React`项目。安装成功后终端执行：

```shel
npm install -g create-react-app
```



## 创建 React 项目

终端`cd`到想创建项目的位置，然后执行：

```shell
create-react-app 项目名称
```
